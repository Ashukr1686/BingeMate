# **App Name**: BingeMate

## Core Features:

- Show Search & Selection: Users can search for TV series by name using the TVMaze API and select a show from the results for analysis.
- Detailed Show Information Display: For the selected show, display key information such as the title, concise summary, total number of episodes, and average episode runtime.
- Total Binge Duration Calculation: Automatically calculate and prominently display the total estimated time (in hours/days) required to watch all episodes of the selected series.

## Style Guidelines:

- Background color: A very dark, desaturated blue-grey (#1B1B26) to provide a sleek, immersive dark theme.
- Primary color: A vibrant, rich blue (#4646F0) to highlight interactive elements, calls to action, and key information, creating a dynamic contrast with the dark background.
- Accent color: A light, cool blue-grey (#CFD8DC), reminiscent of silver, to provide subtle secondary highlighting, distinguish data points, and enhance readability.
- Headline and body font: 'Inter' (sans-serif) for its modern, clean, and highly legible appearance across all text sizes, ensuring a professional aesthetic.
- Utilize sleek, minimalist line icons that align with a modern, data-focused application, enhancing usability without clutter.
- A clean and organized layout that prioritizes content legibility, with a focus on clear presentation of search results and calculated binge durations, using ample spacing.
- Incorporate subtle, smooth transitions and hover effects for search input, result selection, and information display to provide a polished and responsive user experience.
